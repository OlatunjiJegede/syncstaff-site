Security

Your people data, protected by design

HR data is some of the most sensitive data a company holds. Here's exactly how SyncStaff protects it.

Row-level security

Permissions enforced in the database itself. Employees read only their own records; managers only their reports'.

Full audit trail

Every submit, approval, rejection, and role change logged with actor, timestamp, and before/after state.

Encryption

TLS 1.2+ in transit, AES-256 at rest for the database and file storage.

Least privilege

Service credentials never reach browsers; keys are scoped, rotated, and stored in managed secret stores.

Single sign-on

Microsoft Entra ID SSO today; SAML and SCIM provisioning for Enterprise on the roadmap.

Backups & recovery

Automated daily backups with point-in-time recovery on managed Postgres infrastructure.

Infrastructure

SyncStaff runs on Vercel (application hosting) and Supabase (managed PostgreSQL, storage, authentication). Both providers maintain SOC 2 Type II compliance programs.

Responsible disclosure

Found a vulnerability? Email security@syncstaff.app. We acknowledge reports within 48 hours and don't pursue good-faith research.

Data processing

Enterprise customers can request our DPA, sub-processor list, and security questionnaire via sales.
